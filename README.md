## Usage

1. Download file from [release](https://github.com/lion-rion/rac-tester/releases/tag/test)

2. Use command below in the directory where the file exists.

```sh
./rac-test test 
```

if you use simulator, use --sim flag.

```sh
./rac-test test --sim
```
